# Content Strategy — GØL Web Studio

Tone of voice, заголовки, CTA, запрещённые клише и правила UX-копирайтинга для лендинга **golwebstudio.ru**.

Документ для авторов контента и разработчиков, которые меняют тексты в `components/*.tsx`.

---

## Tone of voice

GØL говорит как **спокойная премиум-студия**, не как отдел продаж.

| Да | Нет |
|----|-----|
| Уверенно, без крика | Давление, срочность, FOMO |
| Конкретно | Абстрактные «инновации» |
| Честно | Фейковые цифры и обещания |
| Простой русский | Корпоративный канцелярит |
| Уважение к времени клиента | Инфоцыганство, «бесплатно только сегодня» |

### Голос в одном предложении

*Мы делаем понятные сайты и объясняем, как это работает — без театра и без обещаний «лидер рынка».*

## Как писать заголовки

### Принципы

1. **Коротко** — одна мысль на заголовок.
2. **Конкретно** — что получит клиент, не «комплексные решения».
3. **Без перегруза** — не больше одного highlight-слова (gold span).
4. **Без инновационного мусора** — «инновационные», «передовые», «лидер».
5. **Вопрос или факт** — лучше, чем пустой слоган.

### Хорошие примеры (в духе production)

- Чистый дизайн. Сильный результат.
- Сайт есть. Заявок нет.
- Что мы создаём
- Работаем прозрачно
- Где бизнес теряет деньги
- Как мы работаем
- Результаты
- Обсудим ваш проект

### Плохие примеры

- Инновационные digital-решения для вашего бизнеса
- Лидер рынка веб-разработки
- Создаём уникальные продающие сайты под ключ
- Максимальная эффективность вашего бизнеса

### Структура секции

```
[Label uppercase, 1 строка]     → «Услуги», «Процесс»
[H2, 1–2 строки]                → главная мысль
[Body, 1–2 предложения]         → уточнение без воды
```

## Как писать descriptions

### Да

- Объяснять **пользу**: что станет понятнее, быстрее, проще.
- Писать **простым языком**: «сайт ведёт к обращению», не «оптимизация воронки конверсии».
- Одно предложение = одна мысль.
- Активный залог: «делаем», «показываем», «фиксируем».

### Нет

- «Осуществляем комплексное сопровождение digital-присутствия».
- Три прилагательных подряд: «современный, стильный, уникальный».
- Пустые обещания без механики: «гарантируем результат».

### Шаблон для карточки услуги

```
[Что делаем] + [зачем клиенту] + [что изменится после].
```

Пример (production, services):

> *Структура, оффер и визуальная подача, чтобы с первого экрана было понятно, почему выбрать вас.*

### Шаблон для этапа process

```
[Действие] — [что видит клиент] — [срок если есть].
```

## CTA philosophy

CTA ведёт к **разговору**, не к «заявке в воронку».

### Использовать

| CTA | Контекст |
|-----|----------|
| Обсудить сайт | hero, final-cta |
| Посмотреть кейсы | hero secondary |
| Разобрать задачу | process, cases |
| Отправить заявку | форма (нейтрально, без давления) |
| Написать в Telegram | контакты |

### Не использовать

| CTA | Почему |
|-----|--------|
| Оставьте заявку | канцелярия, давление |
| Получите бесплатную консультацию | инфоцыганский триггер |
| Мы вам перезвоним | обещание без контекста |
| Лучшее решение для бизнеса | пустой claim |
| Заказать продающий сайт | клише + дешёвый маркетинг |
| Узнать цену за 5 минут | fake urgency |

> **Legacy на сайте:** в `solution.tsx` заголовок формы «Получите бесплатную консультацию» — при следующем контент-проходе заменить на «Разобрать задачу» или «Обсудить проект».

## Запрещённые клише

Не использовать в новых текстах (и постепенно вычищать из старых):

### Команда и подход

- команда профессионалов
- индивидуальный подход
- индивидуальный подход к каждому клиенту
- мы — команда экспертов
- только лучшие специалисты

### Рынок и качество

- лидер рынка
- лучший сервис
- лучшее решение
- №1 в …
- максимальная эффективность
- высочайшее качество

### Продукт и услуги

- инновационные решения
- передовые технологии
- сайт под ключ
- продающий сайт
- уникальный дизайн
- эксклюзивный дизайн
- комплексные digital-решения
- полный цикл разработки (допустимо редко, если сразу расшифровать)

### Маркетинговый шум

- увеличим продажи в N раз
- x10 конверсии
- гарантируем результат
- бесплатная консультация (как главный hook)
- только сегодня
- уникальное предложение
- ваш успех — наша цель

### Пустые усилители

- по-настоящему
- максимально
- уникально (без доказательства)
- эксклюзивно
- революционно

## Кейсы

### Как оформлять

- **Коротко:** задача → что сделали → что изменилось словами.
- **Честно:** без выдуманных процентов и «+45% за месяц».
- **Акцент на понятности:** «заявки выше», «предложение яснее» — качественно, не KPI-фейк.
- **Ниша и контекст:** кто клиент, что болело.

### Структура карточки

```
[Название / ниша]
[1–2 строки: было → стало]
[2–3 формулировки результата без цифр-%]
```

### Не писать

- «Конверсия выросла на 340%»
- «Окупился за 2 недели» (без доказательства)
- «Лучший сайт в нише»
- Stock-имена и fake logos

### CTA после кейсов

- «Похоже на вашу задачу?» + «Разобрать задачу» — ок.
- «Хотите так же?» — слабовато, но допустимо; лучше «Разобрать задачу».

---

## Отзывы (testimonials)

- Реалистичный тон: спокойная благодарность, не восторг рекламы.
- Упоминать **конкретику:** сроки, правки, понятность, а не «лучшие на рынке».
- Имя + роль/ниша, без «Иван И., довольный клиент».
- Длина: 2–4 предложения, не простыня.

---

## UX copy (формы, кнопки, ошибки)

### Принципы

- Короткие кнопки (2–3 слова).
- Labels понятные: «Телефон», не «Контакт для связи».
- Placeholder не дублирует label без нужды.
- Ошибки **человеческие**: «Введите номер в формате +7 (999) 999-99-99».
- Success: «Заявка отправлена» + что дальше (мы свяжемся).

### Форма — поля

| Поле | Label / placeholder |
|------|---------------------|
| Имя | Имя |
| Телефон | Телефон / +7 (999) 999-99-99 |
| Сообщение | Кратко опишите задачу (optional tone) |

### Не писать в форме

- «Оставьте контакты и мы перезвоним в течение 5 минут»
- «Обязательное поле для получения скидки»
- Сбор email/Telegram в поле «контакт» (только телефон по продукту)

## Секционные лейблы (uppercase)

Единый стиль:

- `Услуги`, `Процесс`, `Результаты`, `Отзывы`
- tracking широкий, gold или muted
- не: `НАШИ УСЛУГИ ДЛЯ ВАС`

---

## Problem / Solution narrative

### Problem (боли)

- Называть **реальные** ситуации: нет заявок, непонятный оффер, устаревший вид.
- Без запугивания: не «вы теряете миллионы».
- Закрывать блок спокойным «Это решаемо» + срок (2–3 недели) — ок.

### Solution (ответ)

- Не продавать себя как «№1».
- 4 коротких тезиса с тегами (срок, формат, прозрачность).
- Форма — приглашение к диалогу, не «захватите лид».

---

## Trust / FAQ (когда появится)

### Trust

- Факты: договор, этапы, не пропадаем, понятные условия.
- Без «10 лет на рынке» без подтверждения.

### FAQ (план)

- Вопросы как клиент спрашивает: «Сколько стоит?», «Сколько по времени?», «Что нужно от меня?»
- Ответы 2–4 предложения, без воды.

## Brand feeling

GØL должен ощущаться:

| | |
|---|---|
| Спокойно | Нет визуального и вербального крика |
| Дорого | Сдержанный gold, качественная типографика |
| Уверенно | Знаем процесс, не оправдываемся |
| Современно | Next.js, чистый UI, без шаблона 2018 |
| Технологично | Без buzzwords — через дело |
| Без суеты | Нет таймеров, pop-up, «акций» |

### Ассоциации

- digital studio, не «маркетинговое агентство»;
- архитектор сайта, не «менеджер по заявкам»;
- партнёр на проект, не «подрядчик на лендинг за 3 дня».

### Не ассоциировать с

- инфобиз, курсы, «продвижение»;
- дешёвый фриланс-биржа;
- корпоративный интегратор с КП на 40 страниц.
